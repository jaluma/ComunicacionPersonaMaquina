Panel que compone cada producto



Mediante una lista de productos, podemos ojear todo el catalogo que se dispone. Cada producto tiene un panel que le muestra la información referente a él. Consta:

  • Imagen: Todo producto tiene a su disposición una imagen. En caso de que no la tengamos en la base de datos se muestra una por defecto.
  • Nombre: Todo producto viene dado con un formato descriptivo del tipo de producto en cuestión (PAQUETE/ENTRADA/ALOJAMIENTO ).
  • Descripción: Todo producto tiene una breve descripción de sus caracteristicas. Entra ellas se encuentran:
    • Descripción del parque: Breve descripción de las bondades del parque.
    • Lugar: Lugar del mundo donde se encuentra el parque.
    • (Solo alojamiento/paquete) Estrellas: Categoria que tiene el alojamiento en cuestión.
    • (Solo paquete) Nombre del alojamiento: Nombre que tiene el alojamiento incluido en el paquete.
    • (Solo paquete) Duración: Duración del paquete.
  • Panel de tipo: Se incluye una pequeña descripción grafica del tipo de producto:
    • Hotel: Icono de un Hotel. Además se incluirá un boton selector para añadir el desayuno.
    • Apartahotel: Icono de un Hotel y un edificio.
    • Apartamento: Icono de un edificio.
    • Entrada: Icono de parque y entrada.
    • Paquete: Todos los iconos anteriores.
  • Precio: Indica el precio que tiene el producto con los filtros seleccionados.
  • Añadir: Se añade a el carro un producto.
  • (Solo si se encuentra en oferta) OFERTA: Aparece una etiqueta mostrando si un producto está en oferta.
